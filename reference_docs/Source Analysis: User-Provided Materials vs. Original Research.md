## Key Enhancements to Add

- **ARRI Alexa 35** - Modern digital cinema standard

### 2. **Specific Lens Characters** (SIGNIFICANT ENHANCEMENT)

The user materials provide detailed lens profiles with their unique characteristics:

- **Cooke** - "Cooke Look" with unique bokeh
- **Zeiss Ultra Prime** - Sharp, clean spherical optics

### 3. **Aperture Control Detail** (CRITICAL ADDITION)

The user materials provide specific aperture guidance:

- **f/1.4** - Shallow depth of field (sharp subject, blurry background)
- **f/4** - Balanced depth of field for mid-range shots
- **f/11** - Deep focus for landscapes and wide scenes

**My original guide mentioned aperture conceptually but didn't provide these specific f-stop recommendations.**

### 4. **"Light First, Face Second" Philosophy** (PRODUCTION-LEVEL INSIGHT)

This is a **critical enhancement** not explicitly stated in my original guide:

> "AI faces look fake because the light is fake, not because the face is."

This reframes the entire approach to character prompting. The user materials emphasize:
- Specify light source TYPE (soft bounced key, window spill, screen glow)
- Specify light DIRECTION (key from camera-left at 45Â°, top-down)
- Specify contrast ratio (low-key, deep shadows, no fill)

**My guide covered lighting but didn't emphasize this hierarchy.**

### 5. **Negative Prompts** (MAJOR ADDITION)

The user materials extensively cover negative prompts - what to EXCLUDE:

**For Realism:**
- CGI, 3D render, Unreal Engine, Octane render
- Video game, cartoon, anime
- Smooth, pristine, perfect, clean, flawless
- Oversaturated, flat lighting, low contrast
- Watermark, text

**For Faces:**
- No CGI look, no plastic skin, no waxy texture
- No over-sharpening, no artificial symmetry
- No video game character

**My original guide did NOT cover negative prompts at all - this is a critical omission.**

### 6. **Micro-Contrast Control** (TECHNICAL ENHANCEMENT)

User materials introduce the concept of controlling micro-contrast to avoid CGI look:

- "Film-like contrast"
- "Gentle highlight rolloff"
- "Soft tonal transitions"
- "No digital sharpening"

**Avoid:** "hyper-detailed," "8K clarity," "ultra sharp"

**My guide mentioned texture but didn't address micro-contrast specifically.**

### 7. **The Golden Rule** (PRODUCTION PHILOSOPHY)

> "If a real cinematographer couldn't say it on set, don't put it in the prompt."

This is an **excellent guiding principle** that should be prominently featured.
