NANO BANANA PRO PROMPT BREAKDOWN

Achieve nuanced creative control, your prompt must define the story, subject, and style. Include these six variables for maximum impact.

Subject

Who is in the image? Avoid ambiguity and be specific. Don't just say "dog." write: "A Shiba Inu with metallic plating"

Composition

Direct the virtual camera. Specify parameters like "Macro lens for texture," "Isometric view from above," "Fisheye distortion," and etc.

Action

What is the movement? Static images are boring. Define movement and add energy: "Leaping across a rooftop gap"

Location

Where is the scene? Establish the atmosphere. Describe the setting: "A neon-lit Tokyo back alley"

Style

What is the medium? Dictate the artistic output and aesthetics of the image. Choose descriptions like "Vintage 1980s polaroid" and etc.

COMPOSITION AND ASPECT RATIO

Establish rigid canvas boundaries to prevent composition drift. Specify aspect ratios numerically (e.g., 16:9, 2:3) and define the specific shot scale

CAMERA AND LIGHTING DETAILS

Simulate physical photography by defining lens focal length, aperture, and shutter speed to enforce realistic volumetric depth.

SPECIFIC TEXT INTEGRATION

Maximize text legibility by isolating string literals in double quotes. Explicitly define font family to ensure the model renders readable characters

FACTUAL CONSTRAINTS

Enforce logical consistency by using negative constraints to prohibit geometric distortion in charts or biological inaccuracies

REFERENCE INPUTS

When utilizing image inputs, assign specific weight values to control how much influence the uploaded image retains.
